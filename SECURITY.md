# Security Policy

## Supported Versions

The most recent version (v0.x) is supported for security updates

## Reporting a Vulnerability

Please report security vulnerabilities to [homelogger@francislaboratories.com](mailto:homelogger@francislaboratories.com). Minor issues may be submitted via creating a Bug Fix Isssue.

Include the following:
- Where you found the vulnerability
- Where in the codebase it is found
- Your proposal to address it (if any)
